# BTC-PUZZLE-V2.1

## Instruções:

```
0 - CERTIFIQUE-SE DE TER O NODE INSTALADO
1 - ABRA O TERMINAL INSTALE AS DEPENDENCIAS: npm install ou npm i
2 - ABRA O TERMINAL DENTRO DA PASTA E EXECUTE: node main.js

COMO O PROGRAMA FUNCIONA:
 1 - AO INICIAR O PROGRAMA COM 'node main.js' VOCE DEVERÁ ESCOLHER UMA OPCAO DO PUZZLE ENTRE 1 - 160
 2 - EM SEGUIDA VOCE DEVERÁ ESCOLHER ENTREE 3 MODOS DISPONIVEIS:
   1 > Para random ( código rodará do min ao max de forma randomica até econtrar a chave privada ou até voce cancelar a busca)

   2 > Para random com percentual ( código rodará do min ao max de forma randomica até encontrar a chave privada ou até voce cancelar a busca, mas com um percentual de 1% até 99% definido pelo usuario. o valor percentual definido passará a ser a chave mínima da busca)

   3 > Para sequencial com percentual ( código rodará do min ao max de forma sequencial até encontrar a chave privada ou até voce cancelar a busca, porém também é possível definir um valor percentual de 1% até 99% definido pelo usuario. o valor percentual definido passará a ser a chave mínima da busca)

**LEMBRE-SE QUE É NECESSÁRIO TER O NODE INSTALADO NA SUA MÁQUINA**
**NODE VERSÃO 20 OU SUPERIOR
**OBS: todo o conteúdo disponível é apenas para fins educacionais.

```
